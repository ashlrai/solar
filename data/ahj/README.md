# AHJ Knowledge Graph — seed rule sets

Each jurisdiction lives as a single JSON file under `data/ahj/<state>/<slug>.json` conforming to `schema.ts`.

Coverage target:
- **Week 2**: 3 seed jurisdictions (VA-Fairfax, CA-LosAngelesCity, TX-Austin)
- **Month 2**: 15
- **Month 4**: Top 50 US AHJs (~70% of US install volume)

Source material: AHJ websites, SolarAPP+ coverage list, published permit checklists, Shovels.ai exports, direct calls with permit expediters. Every rule MUST cite its source URL + retrieval date.
