---
title: SolarEdge
description: Stream SolarEdge monitoring data into the O&M wedge.
---

Same shape as the [Enphase integration](/api/om/ingest). Set
`inverter_platform: "solaredge"` and use SolarEdge's Monitoring API to fetch
readings on whatever cadence makes sense for your fleet (we recommend
hourly for residential, 5-minute for commercial).
